generated from local activity data with a privacy-first pipeline (aggregate counts only; no
private code, messages, or document text). See `BUILD.md` for how it was built and what was
deliberately excluded.

Served at `/resume/scale-yourself/`. Marked `noindex` — unlisted, not linked from the site,
and kept out of search engines. Static: open `index.html` directly, no build or network needed.
